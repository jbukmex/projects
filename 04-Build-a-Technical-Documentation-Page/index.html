King Rhoam

♫ F♮ ♫

In the long run, if King Rhoam didn't confiscate Terrako when Zelda was young, there wouldn't have been a timline split at all, since it was when Terrako was reactivated that he went back in time.

This is both good and bad because if Terrako stayed activated, he could have helped in the fight against the calamity and Ganon would have never gone back in time, but it is good because it created a timeline split where there was no 100 year apocalypse.
